# Max's 12-Week Powerbuilding Program — Full Reference

## Athlete Profile
- **Experience**: Advanced (3+ years)
- **Gym**: Gymbox Farringdon (full commercial kit)
- **Session length**: 60 minutes max
- **Frequency**: 5 days/week (flexible scheduling, avoid back-to-back lower days)

### Current 1RMs
| Lift | 1RM |
|------|-----|
| Squat | 180kg |
| Bench | 125kg |
| Deadlift | 200kg |
| OHP | 80kg |

### Injury Management
- **Patellar tendonitis**: Low-volume plyos OK (singles/doubles for height). Box squats preferred. Leg extensions kept light.
- **Lower back**: RDLs substituted with GHD hip extensions. Leg press used for quad volume without spinal loading.

### Preferences
- Prefers 5x5 over 4x6
- Box jumps: max height focus (singles/doubles, build up per set, always step down)
- Favourite exercises: SSB squat, box squats, pendulum squat, leg press, incline bench/DB press, hammer strength fly, weighted pull-ups, leg extension, leg curl, GHD, cable/machine work, plyo push-ups, med ball chest throws, DB Bulgarian split squats, seal DB rows
- Design: sleek monochrome (black/white/grey), no emojis

## Program Design Decisions

### Split: 5-Day Hybrid (not pure U/L)
Chose this over Upper/Lower/Accessories because:
- Each of the 4 main lifts (squat/bench/dead/OHP) gets its own day as the priority
- At 180/200kg squat/dead, combining them in one 60-min session is impractical
- No throwaway "accessories only" day that gets skipped

### Day Structure
| Day | Focus | Primary Lift |
|-----|-------|-------------|
| D1 | Lower Strength | Squat variation (rotated by block) |
| D2 | Upper Strength | Bench Press |
| D3 | Pull / Hinge + Core | Deadlift |
| D4 | Lower Hypertrophy + Plyos + Core | Pendulum Squat / Leg Press |
| D5 | Upper Hypertrophy + Plyos + Core | OHP |

### Day 3 Restructure (from original plan)
- Removed leg press from Day 3 (was cluttering a deadlift day)
- Added face pulls (rear delt / rotator cuff, complements heavy pulling)
- Added seal DB rows (replaced cable rows — chest-supported, zero lower back stress)
- Moved leg press to Day 4 (natural home on lower hypertrophy day)
- Result: Day 3 is a clean posterior chain + pull day

### Squat Variation Rotation
| Block | Weeks | Squat Variation | Rationale |
|-------|-------|-----------------|-----------|
| Accumulation | 1-3 | SSB Squat | Upper back strength, quad loading |
| Intensification | 5-7 | Box Squat | Knee-friendly, hip drive |
| Peaking | 9-11 | Back Squat | Specificity for testing |

### Accessories
- Double progression: hit top of rep range for all sets, then add 2.5kg (upper) or 5kg (lower)
- Deload weeks: drop to 2 sets at same weight
- Stay in 3-4 sets x 8-15 rep range throughout

### Arms Volume Decision
- Bumped tricep pushdown and cable curl from 2 sets to 3 sets (Day 2)
- Weekly direct arm work: 3 sets curls + 3 sets pushdowns + all indirect volume from pressing/pulling
- If arms lag after a block, option to add curl/extension superset to Day 5

## Session Details

### Day 1 — Lower Strength
1. Box Jumps — Max Height (4x1-2, build up height, step down)
2. Squat Variation [SSB / Box / Back] (periodized)
3. DB Bulgarian Split Squat (3x8-10 /leg)
4. Leg Curl SS Standing Calf Raise (3x10-12 / 3x12-15)
5. Hanging Leg Raise (3x10-15)

### Day 2 — Upper Strength
1. Med Ball Chest Throw (3x5)
2. Bench Press (periodized)
3. Weighted Pull-Ups (4x5-8)
4. DB Incline Press SS Face Pulls (3x8-10 / 3x15)
5. Tricep Pushdown SS Cable Curl (3x12-15 / 3x12-15)

### Day 3 — Pull / Hinge + Core
1. Deadlift (periodized)
2. GHD Hip Extension (3x10-12)
3. Seal DB Row (3x8-10, per hand, chest-supported)
4. Face Pulls (3x15)
5. Ab Wheel Rollout SS Pallof Press (3x8-10 / 3x10 per side)

### Day 4 — Lower Hypertrophy + Plyos + Core
1. Box Jumps — Max Height (3x1-2)
2. Pendulum Squat (4x10-12)
3. Leg Press (3x10-12)
4. DB Bulgarian Split Squat (3x10-12 /leg)
5. Leg Extension SS Leg Curl (3x12-15 / 3x10-12)
6. GHD Sit-Ups SS Dead Bugs (3x12-15 / 3x10 per side)

### Day 5 — Upper Hypertrophy + Plyos + Core
1. Plyo Push-Ups (3x5-8)
2. OHP (periodized)
3. Weighted Pull-Ups (3x6-10)
4. Hammer Strength Fly SS Cable Rear Delt Fly (3x10-15 / 3x12-15)
5. Cable Lateral Raise SS Cable Woodchop (3x12-15 / 3x10 per side)

## Autoregulation
- If a set hits RPE 10 before planned reps, stop and drop weight 5%
- Deload weeks: RPE 5-6, accessories drop to 2 sets

## Week 12 Test Protocol
- Option A: Deload only (3x3 @ 70%)
- Option B: Work up to new 1RM
  - Warm-up: Bar x10 → 40% x5 → 60% x3 → 75% x2 → 85% x1 → 92.5% x1 → Attempt

## Technical Notes (HTML App)
- File: `powerbuilding.html` — single-file, offline, no dependencies
- All barbell weights pre-calculated from 1RMs, rounded to nearest 2.5kg
- Plate breakdown shown per side (assumes 20kg bar)
- localStorage key: `pb12wk_v3` — saves week/day selection, all set logs, 1RM settings
- 1RM editable via settings modal — all weights recalculate on save
- Monochrome design (black/white/grey, --bg:#111, --card:#1a1a1a), no emojis, mobile-optimized
- Event delegation on #app for efficient DOM handling
- Week has-data check scans log keys (not exercise regeneration) for performance
- **Previous data display**: Each set row shows dimmed "prev:" label with weight/reps from the most recent earlier week that has logged data for that same exercise/set. Scans backwards through all prior weeks, so it works across skipped weeks and deloads. Most useful for accessories (double progression tracking).
